# product-management-app

A Flutter app to manage products and categories with online/offline mode.

## Features
- List products and categories
- Add, update, delete products and categories
- Upload images for products
- Offline mode with local db
- Filter products by category and price

## Tech Stack

- **Frontend:** Flutter, Dart
- **State Management:** Provider
- **Backend:** Laravel (PHP)
- **API:** RESTful API
- **Database:** MySQL (backend), Hive (Flutter offline db)
